Bienvenue sur monoserver!


Cette page qui s'affiche dans votre navigateur se trouve dans le répertoire "www" du serveur. Son nom est "index.html".
Vous pouvez la supprimer et placer, dans ce répertoire "www", d'autres pages HTML ou PHP.


Toute page nommée "index.html" ou "index.php" est automatiquement ouverte par le serveur.

Exemple 1 : "index.html" ou "index.php" dans "www"
            "http://127.0.0.1:1212/" et "http://127.0.0.1:1212/index.html" donnent le même résutlat
Exemple 2 : "index.html" ou "index.php" dans "www/projet"
            "http://127.0.0.1:1212/projet" et ""http://127.0.0.1:1212/projet/index.html" donnent le même résutlat


Pour ouvrir une page particulière (par exemple "bonjour.html"), il faut saisir l'adresse complète dans le navigateur.

Exemple : "bonjour.html" dans "www/projet"
          Pour ouvrir cette page, il faut saisir l'adresse suivante dans le navigateur : "http://127.0.0.1:1212/projet/bonjour.html"
